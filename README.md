# SkillCraft Technologies – Web Development Internship

## Task 01: Responsive Portfolio Website

This project is a single-page responsive portfolio website created as part of
the SkillCraft Technologies Web Development Internship.

### 🔧 Technologies Used
- HTML5
- CSS3
- JavaScript

### ✨ Features
- Fixed navigation bar
- Hover effects on menu items
- Scroll-based navbar style change
- Animated gradient background
- Scroll reveal animations
- Responsive design for mobile and desktop

### 📂 Project Structure
SCT_WD_01/
│── index.html
│── style.css
│── script.js


### 📌 Description
The website presents personal, academic, and technical information in a clean,
structured, and visually engaging layout. It demonstrates fundamental front-end
development skills without using external libraries or frameworks.
